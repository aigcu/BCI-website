​Beginning of year

Beginning data recording process: Using the OpenBCI GUI software select system control panel, then Cyton (live), then serial from dongle, then rename using (firstname-type of data-date taken). Then start with a 1 minute baseline with lights on, then they close their eyes and lights are off, then after a 2s delay of starting recording they are told to open their eyes to the slide and after 30s they close again and wait 10s, this is repeated for every color.
Formatted data was done through selecting b1 then data tab in new excel worksheet, selecting load from csv, selecting file and then the dropdown menu by load to select load into existing, selecting the query on the right to highlight table and uncheck header and banded row, then selecting remove style in the styles menu, the entire table is then dragged up one row, then select the topmost open row to the left of the table scrolling down the shift clicking the bottommost left open column space. Then at the top in the data part of the excel sheet enter in the number of the file being added starting at zero. This process is repeated for every one of the 9 files recorded for each person.

11_19_2025​

This week we figured out that using marker would remove the problem of stopping and starting the stream too much. So starting from this day the new procedure will be starting with eyes closed for 10 and then 10 more for more accurate data. seconds to account for the 2 seconds of wack data. Then first color for 30 seconds using the marker 1 to start and then stop the data collection. Then repeat for each of the numbers and the 0's in between the sectioned off numbers will be considered the baseline ignoring the minute of baseline recording at the beginning. Then after each color sectioned off 10 seconds of closing eyes to switch color and reset brain. Learned that to test data connection use the cyton signal widget and test each connection then wiggle until green. Data will now be uploaded in a text csv file instead of an excel file.
0. eyes closed (20 secs)
1. Mark
2. show color (eyes open) 30 secs
4. Mark
5. Eyes closed (10 secs)
6. Repeat from step 2 for each color
